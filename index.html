Hybrid


Node.js, "Interiors Store":

Server side site which interacts with a MongoDB database and Express, the Node.js server framework.

A shopping site/app using Javascript, HTML and CSS technologies. The site is fully responsive for web, mobile and tablets. The code implements a CRUD system for adding to and deleting from a shopping cart, as well as adding to, deleting and modifying a membership to the site. The site also processes user payments by incorporating a Paypal API for checkout.


PWA, "Pierre Movies":

A movie rental membership app developed with Progressive Web Application technology.

Applying Javascript, HTLM5, CSS3 and Bootstrap, this fully reponsive PWA is a web application that can be “installed” on your system. It works offline when you don't have an internet connection, leveraging data cached during your last interactions with the app. If you launch the apps, they will run in Chrome.

Developed with Flutter, Dart and Material Design technologies, this app implements list views for exercises by category, and list views for exercises in each of those categories. The app also implements a CRUD system to add, modify and delete from a network contact list.

IOS Swift UIKit, "Hear Me":

"EarsOn" is an app project developed for the hearing impaired. The interfaces above were developed with Swift UIKit.

This app assists the hearing impaired in their every day lives. Using the AVSpeachSynthesizer class, the app produces speech from text utterances as well as transcribes a speaker's voice to text. Using the Swift Core Data framework, the app can add the user's desired voice text shortcuts for different locations, such as a restaurant or a pharmacy. The user can modify the language and accent of voice synthesizing for the app in the settings, coded with the User Defaults class.

IOS Swift UIKit, "MVC Game Project":

This MVC game app was developed as part of course material for learning the fundamentals of Swift. It incorporates several functions, graphics and animations.

Using the model view controller development architecture prefered by Apple, the game label shows questions, which the user must answer with a boolean right swipe for correct, and left swipe for false. Using Swift CGAffineTransform functions, the app incorporates multiple animations: the quetion label pivots right or left at an angle, the score rotates in 3d at the answer of every question, among others. The app also keeps track of the user's high and low score for a current game session, and alerts the user when he is halfway in a 10 question game if the score can be improved or worsened. A slider above the question label lets the user increase or decrease the font size of the questions text. The user can also modify settings for the game on the homepage, such as the number of questions in each game set, and the category of questions in each game set. The questions are provided by an API.

About
